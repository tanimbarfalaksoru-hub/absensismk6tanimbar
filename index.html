Zona Berbahaya (Factory Reset)

Tindakan ini akan menghapus semua data (Siswa, Kelas, Absensi, Pengaturan) yang tersimpan di memori lokal browser ini secara permanen. Data tidak dapat dipulihkan.

Mode Penyimpanan Lokal

Aplikasi ini saat ini berjalan 100% menggunakan LocalStorage Browser. Semua interaksi Create, Read, Update, Delete (CRUD) diproses instan tanpa perlu koneksi internet. Sangat cocok untuk mode Demo atau penggunaan di satu perangkat terdedikasi.
